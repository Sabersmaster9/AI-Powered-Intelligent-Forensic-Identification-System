# AI-Powered Forensic Identification Assistant (RAG Module)

本模块是基于开源项目 [RAGFlow](https://github.com/infiniflow/ragflow) 进行深度二次开发的司法鉴定增强系统。

## ⚖️ 项目背景与贡献
本项目针对司法鉴定（特别是工伤伤残判定）中标准条文繁杂、文书书写效率低等痛点，在 RAGFlow 的基础上进行了以下核心改进：

* **垂直领域知识库定制**：集成了详尽的司法判定依据、伤残鉴定标准及历史典型案例。
    * *注：由于司法数据的敏感性与特殊性，知识库原始数据目前采取闭源处理。*
* **司法文书自动化工作流**：全新搭建了从“鉴定信息录入”到“自动化文书生成”的定制化工作流，实现了鉴定意见书的初稿自动润色与生成。
* **硬件适配优化**：针对本地化部署需求，重新配置并优化了 GPU 计算资源调度（显存分配与并发处理优化）。

---

## 🛠️ 技术实现说明

### 1. 知识库增强 (Knowledge Base)
利用 RAGFlow 强大的文档解析能力，我们构建了针对司法文书的专属解析方案，确保对《劳动能力鉴定 职工工伤与职业病致残等级》等规范性文档的精准检索。

### 2. 工作流 (Workflow)
通过自定义工作流引擎，系统能够提取用户录入的姿态估计数据（由本项目 Pose 模块提供）与病历描述，自动比对知识库中的判定条款，生成结构化的鉴定报告。

### 3. 环境配置修改
为了确保在本地高性能工作站上稳定运行，主要对以下配置文件进行了调整：
- 修改了 `docker-compose.yml` 中的 GPU 映射参数。
- 优化了本地 LLM 接口（如 Ollama/Qwen）的响应延迟。
